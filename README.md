# KAPAAN 🚀
## *AI-Powered Accident Detection & Emergency Response System*

Kapaan is an AI-driven solution designed to detect road accidents in real time, ensure faster emergency response, and minimize traffic disruption. Built as part of the UYIR Road Safety Hackathon 2025, this project secured the *Best Performing Team Award* 🏆 and a *₹10,000 cash prize* for its innovative and cost-effective approach to road safety.

---

## 🌟 Features
✅ *Real-Time Accident Detection* – Uses AI-powered Computer Vision to identify accidents.  
✅ *Emergency Response System* – Sends instant alerts to authorities & medical teams.  
✅ *Traffic Signal Automation* – Automatically switches signals for ambulances to reduce delays.  
✅ *Mobile App Integration* – Allows users to report incidents with live location tracking.  
✅ *Zero-Cost Implementation* – Built using existing infrastructure for cost-effective deployment.  

---

## ⚙ Tech Stack
🔹 *AI & Deep Learning* – Computer Vision models for accident detection.  
🔹 *Python & FastAPI* – Backend server for processing & API integrations.  
🔹 *Flutter* – Cross-platform mobile app for emergency response & reporting.  
🔹 *Firebase* – Real-time notifications & cloud storage.  
🔹 *OpenCV & TensorFlow* – Image processing & AI-based event recognition.

## 🚀 How It Works
1️⃣ *AI Model detects an accident* using real-time footage.  
2️⃣ *System verifies & sends alerts* to emergency services.  
3️⃣ *Traffic signals adjust automatically* for ambulances.  
4️⃣ *Authorities receive real-time notifications* for rapid intervention.

## 🛠 Installation & Setup
### Clone the repository:
bash
git clone https://github.com/your-username/Kapaan.git
cd Kapaan


### Run the backend server:
bash
pip install -r requirements.txt
uvicorn main:app --reload


### Run the mobile app (Flutter):
bash
flutter pub get
flutter run
